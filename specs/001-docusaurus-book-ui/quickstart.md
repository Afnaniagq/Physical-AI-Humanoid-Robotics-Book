# Quickstart: Running the Development Server

This guide provides instructions for setting up and running the Docusaurus development server to preview the textbook website.

## Prerequisites

- Node.js (v16.14 or later)
- npm or yarn

## Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Afnaniagq/Physical-AI-Humanoid-Robotics-Book.git
    cd Physical-AI-Humanoid-Robotics-Book
    ```

2.  **Install dependencies**:
    From the `frontend` directory, install the necessary npm packages.
    ```bash
    cd frontend
    npm install
    ```

## Running the Development Server

1.  **Start the server**:
    From the `frontend` directory, run the following command:
    ```bash
    npm run start
    ```

2.  **View the site**:
    The development server will start, and you can view the website by navigating to [http://localhost:3000](http://localhost:3000) in your web browser. The site will automatically reload when you make changes to the source files.
